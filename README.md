# 🏆 Bolão Online - Copa do Mundo 2026

Sistema de apostas desenvolvido para integração entre colegas de trabalho.

## 🛠️ Tecnologias
- **Python** (Streamlit)
- **Supabase** (PostgreSQL)
- **Pandas** (Análise de Dados)

## 🚀 Como rodar o projeto
1. Clone o repositório
2. Crie o venv e instale os requirements
3. Configure o arquivo .env com suas chaves do Supabase
4. Execute `streamlit run app.py`
